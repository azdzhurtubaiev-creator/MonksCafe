\# Monk's Cafe



Console application for managing a cafe bill (KhAI summer practice).



\## Features

\- Add item (description 3-20 chars, price > 0, max 5 items)

\- Remove item by number

\- Tip as percentage or fixed amount

\- Display bill: Net, Tip, GST (5%), Total

\- Clear all

\- Save/load bill to CSV file



\## Architecture

\- `BillService` — business logic, independent from UI

\- `Ui` — console input/output

\- `Program` — entry point with menu loop



\## Run

dotnet run
